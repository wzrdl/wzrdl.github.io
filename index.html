High-performance B+Tree database engine

I designed and implemented a complete mini-RDBMS in C++ featuring an order-3 B+Tree storage engine with buffer pool management, secondary indexes, and write-ahead logging (WAL). The system includes an interactive SQL shell supporting 15 SQL commands with O(log n) query latency, demonstrating deep understanding of database internals and efficient data structures.

GPU-accelerated kernel K-means clustering

I refactored kernel K-means algorithms to leverage sparse linear algebra operations (SpMM/SpMV) and optimized for GPU execution using CUDA and cuSPARSE libraries. Through careful memory throughput optimization and occupancy tuning, I achieved remarkable performance gains: 1000× faster than CPU and 2.6× faster than dense GPU implementations on MNIST and CIFAR-10 datasets.

AI-powered medical prompt optimization

I developed an advanced prompt optimization system using text-gradient techniques with momentum and Bayesian reverse validation. The system significantly improved performance on medical AI benchmarks, achieving 20% better accuracy on MedQA/PubMedQA compared to Chain-of-Thought approaches. I integrated hybrid retrieval combining BM25 and dense embeddings with cross-encoder reranking for enhanced relevance.
