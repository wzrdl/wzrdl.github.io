Smarter Doctor Agent is a real-time voice-based medical assistant built as a cloud application.

Highlights:

- Built a FastAPI backend and Next.js frontend.
- Integrated Twilio, Elasticsearch, BigQuery, and Gemini through Vertex AI.
- Deployed with Docker and GitHub Actions CI/CD.
