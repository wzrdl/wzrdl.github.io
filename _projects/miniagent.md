Miniagent is a modular training framework for studying tool-augmented agents with reinforcement learning.

Highlights:

- Built a GRPO-based agent training loop for tool-augmented tasks.
- Designed a tool-call interface that supports external web search during rollouts.
- Focused on end-to-end agent RL experiments and evaluation.
