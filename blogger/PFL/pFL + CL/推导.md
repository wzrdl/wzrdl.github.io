这句话后面这一段（online Laplace approximation）主要在讲：**把上一轮学到的后验 (p(\theta!\mid!D_{1:k-1})) 用“在 MAP 点附近的二次近似”压成一个高斯分布**，这样你在学第 (k) 个任务时，就等价于在新任务损失上加一个“把参数拉回旧解”的二次正则项——这就是很多正则化方法（尤其 EWC 一类）的贝叶斯来源。

下面按原段落的推理顺序完整解读（含关键公式推出来）：

---

## 1) 它在做什么近似？

目标：上一轮后验不可解
[
p(\theta\mid D_{1:k-1})
]
于是用一个可处理的近似分布 (q_{k-1}(\theta)) 来代替它。

**online Laplace 的选择**：令 (q_{k-1}(\theta)) 是一个多元高斯：
[
q_{k-1}(\theta)=\mathcal N(\theta;\mu_{k-1},\Lambda^{-1}_{k-1})
]
其中：

* (\mu_{k-1})：上一轮后验的 **mode/MAP**（最可能的参数点）
* (\Lambda_{k-1})：高斯的 **precision matrix**（协方差的逆），刻画“哪些方向很敏感/重要”。

> 文中 “with local gradient information” 的意思是：
> 这个高斯不是凭空来的，而是靠 **在 (\mu_{k-1}) 附近的局部二阶信息（梯度/海森矩阵）** 构造的——也就是二阶泰勒展开。

---

## 2) 为什么二阶泰勒展开会得到高斯？

对 log posterior 在 (\mu_{k-1}) 附近做二阶展开。令
[
\ell_{k-1}(\theta) := \log p(\theta\mid D_{1:k-1})
]

在 (\mu_{k-1}) 处展开：
[
\ell_{k-1}(\theta)\approx \ell_{k-1}(\mu_{k-1})
+(\theta-\mu_{k-1})^\top \nabla \ell_{k-1}(\mu_{k-1})
+\frac12(\theta-\mu_{k-1})^\top \nabla^2 \ell_{k-1}(\mu_{k-1})(\theta-\mu_{k-1})
]

但 (\mu_{k-1}) 是 mode/MAP，所以
[
\nabla \ell_{k-1}(\mu_{k-1}) = 0
]

于是：
[
\ell_{k-1}(\theta)\approx \text{const}
+\frac12(\theta-\mu_{k-1})^\top \nabla^2 \ell_{k-1}(\mu_{k-1})(\theta-\mu_{k-1})
]

因为在极大点附近 (\nabla^2 \ell) 是负定的，所以定义
[
\Lambda_{k-1} := - \nabla^2 \ell_{k-1}(\mu_{k-1})
]
就得到
[
\ell_{k-1}(\theta)\approx \text{const}
-\frac12(\theta-\mu_{k-1})^\top \Lambda_{k-1}(\theta-\mu_{k-1})
]

两边取指数，就得到一个高斯形状：
[
p(\theta\mid D_{1:k-1})\approx
\exp\Big(-\frac12(\theta-\mu_{k-1})^\top \Lambda_{k-1}(\theta-\mu_{k-1})\Big)
\propto \mathcal N(\mu_{k-1},\Lambda^{-1}_{k-1})
]

这就是 “Laplace approximation”。

---

## 3) 用这个近似，学第 (k) 个任务会变成什么优化问题？

贝叶斯递推（你前面问的式 (1)）：
[
p(\theta\mid D_{1:k}) \propto p(\theta\mid D_{1:k-1}),p(D_k\mid \theta)
]

取对数：
[
\log p(\theta\mid D_{1:k})
= \log p(D_k\mid \theta)+\log p(\theta\mid D_{1:k-1})+\text{const}
]

把 (\log p(\theta\mid D_{1:k-1})) 用上面的二次近似替换：
[
\log p(\theta\mid D_{1:k})
\approx \log p(D_k\mid \theta)
-\frac12(\theta-\mu_{k-1})^\top \Lambda_{k-1}(\theta-\mu_{k-1})
+\text{const}
]

因此，求第 (k) 轮的 MAP（文中写的 posterior mode）：
[
\mu_k=\arg\max_\theta \log p(\theta\mid D_{1:k})
]
等价于：
[
\mu_k=\arg\max_\theta \Big[
\log p(D_k\mid \theta)
-\frac12(\theta-\mu_{k-1})^\top \Lambda_{k-1}(\theta-\mu_{k-1})
\Big]
]

**直觉**：新任务想把 (\theta) 往新最优推；二次项把 (\theta) 拉回旧的 (\mu_{k-1})，并且拉回强度由 (\Lambda_{k-1}) 决定（重要方向更不许动）。

---

## 4) 这段话和 EWC 的关系（你读下去马上会看到）

* 如果把 (\Lambda_{k-1}) 近似成对角，并用 Fisher 信息近似它（常用做法），二次项就变成 EWC 那种
  [
  \sum_i \lambda_i(\theta_i-\mu_{k-1,i})^2
  ]
* 所以 survey 这里是在告诉你：**“正则化保护旧知识”可以被解释成“贝叶斯后验的 Laplace 近似”**。
